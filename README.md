# Multiplicative-error-based Three-Cornered Hat (MTCH) Method

MTCH code
The Multiplicative-error-based Three-Cornered Hat (MTCH) Method calculates the root-mean-squared-error (RMSE)  between three or more sets of data (from different measurement systems) and the true.
